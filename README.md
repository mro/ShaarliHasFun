
ShaarliHasFun is the successor of http://mro.name/shaarligo, the
personal, 0-dependency, friendly microblogging engine.

This from-scratch rewrite aims for

* be as easy and ubiquitous to install and operate as before,
* no GAFAM inside,
* hold up to djb quality standards,
* showcase functional programming on the tiniest hardware,
* be a pleasure to use.

# What has to be done

* de-/serialise posts
* de-/serialise configuration
* de-/serialise volatile runtime conditions (blocked aceess)
* create Atom feeds, lazy!
* authenticate
* web interafce
* search
* web interface
* images
* comments
